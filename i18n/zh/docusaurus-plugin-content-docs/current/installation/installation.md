[管理 Server 角色](server-roles.md)详细介绍了如何使用专用 `control-plane` 或 `etcd` Server 设置 K3s。

[管理打包组件](packaged-components.md)详细介绍了如何禁用打包组件，或使用自动部署清单安装你自己的组件。
