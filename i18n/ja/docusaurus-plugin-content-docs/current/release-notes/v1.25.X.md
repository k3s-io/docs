以前のリリースからアップグレードする前に、Kubernetesの[緊急アップグレードノート](https://github.com/kubernetes/kubernetes/blob/master/CHANGELOG/CHANGELOG-1.25.md#urgent-upgrade-notes)を必ずお読みください。
